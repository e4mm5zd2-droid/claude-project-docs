# 進捗更新: infra
*日時: 2026-04-08*
*発話者: taka/matsumoto*

## 要約
複数のWeb3・DeFi関連ツールを並行開発中。AIツール（Claude等）を活用したUI/バックエンド構築フローが中心。広告・マーケティング領域での連携可能性について議論が発生した。

## 進捗・変更点
- **開発中プロダクト（複数並行）:**
  - **エアドロップチェッカー**: ウォレット接続により、未請求エアドロップの現在価値を表示する機能
  - **ファーミング／ステーキング情報サイト**: 外部サイトから流入するおすすめDeFiサービス（3サイト）を表示する機能
  - **マルチチェーン対応DEX連携**: 別DEXおよびEVM系チェーン向けの抜き取りシステムとの統合
  - **LINEステップ（Lステップ）連携**: ボス会社自社開発のLINE問い合わせ自動化ツール。現在公開準備中
- **開発フロー**: Cursor等でUIをほぼ構築 → バックエンドフローを完成 → Claudeに移植・組み込む、という流れで進行中
- **Polymarket / 外部DEXのコピートレード機能**についてアイデアが上がっているが、現状リソース不足で着手できていない
