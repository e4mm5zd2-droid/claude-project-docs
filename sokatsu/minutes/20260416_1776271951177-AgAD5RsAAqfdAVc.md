# ミーティングメモ: 有料職業紹介事業の戦略・方針整理
*日時: 2026-04-16*
*音声ファイル: 1776271951177-AgAD5RsAAqfdAVc.m4a*

## 要約
有料職業紹介の許可申請を資本金500万円がある今のタイミングで進める方針。ただし、事業拡大には求人費・人件費・事務所など多くのコストが必要なため、現時点では積極的な拡大は現実的でない。当面のゴールは「許可取得」と「事務所確保」に絞り、紹介先は直接ではなく中間者を通じた工場紹介を活用する。

## 議題

### 1. 有料職業紹介の仕組みと現状の課題
- 紹介先は主に工場（ライン工・トヨタ系など）が中心
- 振り先（工場）を直接は保有していないため、第三者を介した紹介が必要
- 直接取引先を持つには、一定の紹介件数・人員・広告投資が必要で、現状のキャパでは対応不可

### 2. 事業拡大に必要なコストと現実的な制約
- 求人費が高額（50万円以上）で、Indeed等の媒体費用も別途かかる
- 人員は業務委託では難しく、正社員採用が必要になる見込み
- 求人費・採用費・事務所費用など多面的にコストが発生するため、現時点での積極拡大は困難
- 毎月20〜30件の工場紹介を安定させるには相応の資金力が前提

### 3. 資金調達・キャッシュフローの見通し
- 事業拡大のための運転資金はウォーターサーバー事業のキャッシュフローで賄う想定
- ウォーターサーバー事業が最もキャッシュインが安定しているため、資金源として位置付ける

### 4. 申請の目的と今回のゴール
- 申請の主な理由は「資本金500万円がある今しか申請できない」というタイミングの問題
- 事業を今すぐ本格展開したいわけではなく、**許可と事務所の確保がゴール**
- 戦略的に、求人・採用への投資は現時点では不要と判断

- [x] 工場への紹介は直接ではなく、第三者（仲介者）を通じて行う
- [x] キャッシュ源はウォーターサーバー事業を充てる
