# ミーティングメモ: ショート動画事業の方向性検討

*日時: 2026-04-16*
*音声ファイル: 1776272399265-AgAD7BsAAqfdAVc.m4a*

## 要約
ネイルサロン向けのCRM・LINE公式運用は顧客ニーズとマッチしないと判断し、ショート動画制作サービスへの軸足移行を検討した。AIによるアルゴリズム分析と、キャラクター（擬人化）を活用した汎用的な動画制作を強みとする方向性が浮上した。ターゲットはネイルサロン限定でなく全業種への展開を視野に入れる。

---

## 議題

### 1. CRM・LINE公式運用の方針見直し（ネイルサロン向け）
- ネイリストが顧客と個人DM・LINEで直接やり取りしており、LINE公式での顧客管理は不要との結論
- 顧客は「来たいとき」に自発的に連絡するため、CRMで来店動機をコントロールするのは難しい
- CRM売り込みよりも、別のフックで顧客課題に応えるアプローチが必要

### 2. ショート動画制作サービスの可能性検討
- 月5万円程度でのショート動画運用・制作サービスが、ネイルサロン・美容クリニック等に汎用展開できると判断
- 企画・撮影・編集を担当できる人材が既に確保できている（社内または業務委託）
- 動画の完全AI自動生成は現状困難だが、画像5枚をAIでつなぎ合わせた20秒ショート動画（Kling AI等）は実現可能
- AI動画生成より音声AI分野の方が技術的進歩が進んでいる現状

### 3. AIを活用したアルゴリズム分析の差別化戦略
- TikTok・X・YouTubeショートのアルゴリズムをAIで解析し、効果的な投稿戦略を提案することが競合優位になりうる
- 「AIでアルゴリズムを分析できる」点をOnTheEdgeの強みとして打ち出す方向
- 動画制作はラスト1マイルとして業務委託先に任せる構成も検討

### 4. キャラクター（擬人化）活用型ショート動画フォーマットの検討
- 実在の人物を登場させるより、AIで生成したキャラクター（擬人化した野菜・ゴルフクラブ・ボールなど）に人格を与えるフォーマットが有効
- 理由：汎用性が高い・制作工程が少ない・実写との差別化ができる
- 参考イメージ：memeキャラクター（カエルなど）がトレーダーの日常を演じる動画スタイル
- 「人が出る必要はない」との共通認識

### 5. ターゲット業種の拡張
- 当初はネイルサロン・美容クリニック向けを想定していたが、ショート動画需要は全業種に存在
- ウォーターサーバー営業会社など業種を問わず展開可能と判断
- **ネイルサロン特化から「ショート動画サービス全般」へと議論の軸を移すことで合意**

---

## 決定事項
- [x] CRM・LINE公式運用のネイルサロン向け提案は積極推進しない
- [x] ショート動画制作サービスを主力サービスとして検討を進める
- [x] ターゲットをネイルサロンに限定せず、全業種対応の汎用サービスとして設計する
- [x] キャラクター擬人化フォーマットを軸にしたショート動画の方向性を採用する
